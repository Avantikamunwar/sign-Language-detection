# Sign Language Detection (Real-Time)

This project detects hand gestures for sign language using:
- MediaPipe for hand landmark extraction
- TensorFlow/Keras custom CNN classifier
- OpenCV for real-time video processing

## Features
✔ Collect dataset using hand landmark tracking  
✔ Generate `data.pickle` using create_dataset.py  
✔ Train classifier using trainclassifier.py  
✔ Real-time inference using inferenceclassifier.py  
✔ Smooth performance and low-latency predictions
